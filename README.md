# HandyLinuxUtilities

A number of simple tools to copy to ```/usr/local/bin/``` or just run directly.

## TOOLS

* showenv.pl - Shows the environment variables in an easy-to-read format.
* cpu_info.pl - Shows CPU information or outputs the number of cores.
* awesome.sh - A one-time script to install the Awesome Fonts package that make coding easy and ANSI output with full special character list.
